# Elevie Travel Suite

Starter scaffold for Elevie Travel Suite, a mobile-first internal travel quoting platform.

## Included
- Next.js App Router scaffold
- Tailwind styling
- Mobile-first dashboard shell
- Quote editor mock UI
- Supabase SQL schema
- RLS starter policies
- Environment template

## Next steps
1. Create the GitHub repo
2. Push these files
3. Create a Supabase project
4. Run the SQL migrations
5. Add environment variables
6. Deploy to Vercel or preferred host
